# 教职工管理系统
这是本人大一时候的c语言大作业
这是一个使用C++面向对象编程实现的教职工管理系统，可以管理学校中的不同类型人员（普通员工、教师和管理人员）的信息。

## 功能特点

- **添加教职工**：支持添加不同类型的教职工（普通员工、教师、管理人员）
- **显示教职工信息**：展示所有教职工的详细信息
- **删除教职工**：根据ID删除指定教职工
- **修改教职工信息**：更新教职工的基本信息
- **查找教职工**：根据ID或姓名查找教职工
- **排序教职工**：对教职工列表进行排序
- **数据持久化**：将数据保存到文件中，程序重启后可恢复数据
- **清空数据**：清空所有教职工数据

## 系统架构

系统采用面向对象设计，主要包含以下类：

- **Worker**：抽象基类，定义了所有教职工的共同属性和方法
- **Employee**：派生类，表示普通员工
- **Teacher**：派生类，表示教师
- **Manager**：派生类，表示管理人员
- **WorkerManager**：管理类，负责对所有教职工进行增删改查等操作

## 使用说明

1. 运行程序后，会显示主菜单
2. 输入对应的数字选择功能：
   - 0: 退出系统
   - 1: 添加教职工
   - 2: 显示教职工信息
   - 3: 删除教职工
   - 4: 修改教职工信息
   - 5: 查找教职工
   - 6: 排序教职工
   - 7: 清空数据
